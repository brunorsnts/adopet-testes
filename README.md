## 💻 Sobre o projeto

Adopet é um site fictício de adoção de pets, com funcionalidades para cadastro de tutores, de abrigos e de pets, além de solicitação de adoções. Nesse repositório o projeto será uma API Rest em Java do Adopet.

Este repositório é utilizado como projeto base para o estudo de **testes automatizados com JUnit e Mockito**. O objetivo é aprender a escrever testes de unidade e de integração sobre uma API REST real, cobrindo services, validações e controllers.

---

## ⚙️ Funcionalidades

- [x] Cadastro/atualização de tutores;
- [x] Cadastro de abrigos;
- [x] Cadastro de pets do abrigo;
- [x] Listagem de pets disponíveis para adoção;
- [x] Solicitação de adoção;
- [x] Aprovação/reprovação de adoção;

## 🛠 Tecnologias

As seguintes tecnologias foram utilizadas no desenvolvimento da API Rest do projeto:

- **[Java 17](https://www.oracle.com/java)**
- **[Spring Boot 3](https://spring.io/projects/spring-boot)**
- **[Maven](https://maven.apache.org)**
- **[MySQL](https://www.mysql.com)**
- **[Hibernate](https://hibernate.org)**
- **[Flyway](https://flywaydb.org)**

Tecnologias utilizadas nos testes:

- **[JUnit 5](https://junit.org/junit5/)** — framework de testes para Java
- **[Mockito](https://site.mockito.org/)** — criação de mocks e verificação de comportamento
- **[Spring Boot Test](https://docs.spring.io/spring-boot/docs/current/reference/html/test-auto-configuration.html)** — suporte a testes de integração com contexto Spring
